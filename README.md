# Vimium-options

This repository contains a basic Vimium setup for customizing and restoring
browser plugin behavior and appearance, using the Catppuccin color scheme
(Mocha flavor) with styling choices that avoid an overwhelming blue look.
The vimium-ui.css file is included to make hint marker style edits simple and
direct. The vimium-options.json file is a direct backup exported from Vimium
settings, so it can be imported to quickly restore configuration.
